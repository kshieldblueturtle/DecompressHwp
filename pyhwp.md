# pyhwp 설치 
https://github.com/mete0r/pyhwp <br/>

<br/>

# anaconda  가상환경 설치
```
$ conda create -n pyhwp python=2.7 
```

<br/>

# anaconda 가상환경 활성화
```
$ source activate pyhwp
```

<br/>

# pyhwp 테스트
```
./hwp5html --output=~/test/output ~/test/sample/sample.hwp
```
